Apa itu Brizy?

Sebelum membahas bagaimana cara membuat website gratis dengan Brizy, saya ingin mengulas sedikit mengenai apa itu Brizy. Brizy adalah sebuah alat website builder, lebih dikenal sebagai plugin WordPress sama seperti elementor namun Brizy tidak setenar elementor. Brizy juga memiliki fitur SAAS Cloud Website Builder dimana kamu bisa membuat website disana tanpa perlu memiliki hosting seperti WIX, nahh pada update terakhir kini brizy cloud meluncurkan fitur blogging, ini membuat brizy cloud menjadi CMS (Content Management System). Pada roadmapnya di masa depan brizy cloud juga akan memiliki fitur custom post type sehingga kemungkinan brizy cloud akan seperti Webflow.
